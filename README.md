# Marketing Performance Dashboard (Power BI)
## Overview
Marketing teams often invest significant budgets across multiple advertising channels, customer segments, and locations. The challenge is identifying which campaigns generate the highest return while minimizing advertising costs.
This dashboard helps decision-makers monitor campaign performance and optimize future marketing strategies using data-driven insights.
## Business Objectives
This dashboard answers questions such as:
- Which advertising channels deliver the highest ROI?
- Which customer segments achieve the highest conversion rates?
- Which locations and languages perform best?
- How do CPC and CTR affect campaign performance?
- Are there campaigns with high spending but poor returns?
- How does campaign performance change over time?
